# AnimationTools
UE5.5下的动画序列分割工具，可以将一整个动画序列按帧分割成多个，目前只进行了关键帧的复制，曲线、通知等都还没处理
使用说明：在内容浏览器中选中一个序列，右键单击弹出菜单，执行其中的“分割动画序列”，即可弹出分割动画序列的窗口进行分割操作，分割后的多个片段缺省会保存在原动画序列所在目录的Anims子目录中
